projects

ELK stash

Utilizando máquinas virtuales en la nube, desarrollé un proyecto centrado en DFIR, simulando ataques y defensas mientras centralizaba la recolección de eventos con ELK Stack. Este enfoque me permitió crear un entorno seguro y controlado para analizar incidentes de manera efectiva.

Cloud Resume Challenge

Este proyecto forma parte del Cloud Resume Challenge, donde desplegué esta página web utilizando AWS. Configuré DNS, desarrollé funcionalidades en JavaScript y Python, e integré una API con una base de datos. Implementé CI/CD con GitHub para automatizar el despliegue y utilicé Infrastructure as Code (IaC) para gestionar el entorno de manera eficiente, junto con control de versiones para asegurar un flujo de trabajo organizado.

Malware analysis lab

Laboratorio de Análisis Forense Digital con FlareVM. Implementé un entorno de análisis de malware personalizado para investigar incidentes de seguridad en profundidad. Utilicé FlareVM como base y lo extendí con herramientas de análisis de memoria, como Volatility, para investigar incidentes de memoria. A través de este proyecto, desarrollé habilidades en la creación de líneas de tiempo de eventos, la identificación de indicadores de compromiso (IoCs) y la generación de informes forenses detallados.

Active Directory

En este proyecto configuré un entorno de Active Directory en múltiples máquinas virtuales, tanto on-premises como a través de una VPN. Aprendí a centralizar eventos en un SIEM y a generar telemetría para el análisis de ataques. Utilicé herramientas como Windows, Kali Linux, Splunk y Atomic Red Team para simular amenazas y monitorear la seguridad del entorno.
